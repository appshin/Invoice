# KOWOO 인보이스 QR 라벨 발행

Packing list(인보이스) 엑셀 파일을 올리면 **(8) No. & Date of invoice** 항목의
인보이스 번호와 날짜를 자동으로 읽어 **QR 라벨을 원하는 매수만큼 인쇄**하는 웹앱입니다.

서버 없이 브라우저에서만 동작하므로 GitHub Pages에 그대로 올려서 쓸 수 있습니다.
**엑셀 파일은 어디에도 업로드되지 않고 브라우저 안에서만 읽습니다.**

## 사용 흐름

1. 엑셀 파일(.xlsx)을 드래그하거나 클릭해서 선택
2. 인보이스 번호·날짜가 자동으로 채워짐 (필요하면 직접 수정 가능)
3. 총 매수 / 시작 번호 / 인쇄 매수 입력
4. 미리보기 확인 → **인쇄**

## QR 중복 방지 규칙

각 라벨의 QR에는 **일련번호**가 들어가므로 내용이 전부 다릅니다.

```
KWINV|KW260716-RM|Jul-16-2026|01/20
KWINV|KW260716-RM|Jul-16-2026|02/20
KWINV|KW260716-RM|Jul-16-2026|03/20
                              ↑ 이 부분이 라벨마다 달라짐
```

| 항목 | 내용 |
|------|------|
| `KWINV` | 인보이스 라벨 식별자 (스캐너에서 이 값으로 종류 구분) |
| `KW260716-RM` | 인보이스 번호 (엑셀에서 자동 추출) |
| `Jul-16-2026` | 발행일 (엑셀 원본 서식 그대로) |
| `01/20` | **일련번호 / 총 매수** — 라벨마다 고유 |

**재인쇄(라벨 손상 교체) 시**: 시작 번호를 그 번호로, 인쇄 매수를 1로 두면
**원래 라벨과 완전히 같은 QR**이 나옵니다. 같은 파렛트를 가리키는 라벨이므로
새 번호가 생기지 않아야 재고가 이중으로 잡히지 않습니다.

## 인보이스 자동 인식 방식

1. 시트에서 **"No. & Date of invoice"** 라벨 문구를 찾아 **바로 아래 행**에서
   인보이스 번호(문자열)와 날짜(날짜 셀)를 읽습니다.
2. 못 찾으면 고정 위치 **G4 / J4** 를 확인합니다.
3. 그래도 못 찾으면 화면에 안내가 뜨고 직접 입력할 수 있습니다.

양식이 조금 바뀌어도 라벨 문구만 있으면 계속 동작합니다.
날짜는 엑셀 서식(`mmm-d-yyyy`)과 동일하게 `Jul-16-2026` 형태로 표시됩니다.

## 라벨 형식

```
┌──────────────────────────────────────────────┐
│  ┌────────┐   INVOICE NO.                    │
│  │        │   ┏━━━━━━━━━━━━━━━┓        NO.   │
│  │   QR   │   ┃  KW260716-RM  ┃         03   │
│  │        │   ┗━━━━━━━━━━━━━━━┛        / 20  │
│  │        │   DATE                           │
│  └────────┘   ┏━━━━━━━━━━━━━━━┓              │
│               ┃  Jul-16-2026  ┃              │
│               ┗━━━━━━━━━━━━━━━┛              │
└──────────────────────────────────────────────┘
     QR          노란 하이라이트 = 육안 식별용
```

- **스캔이 안 되는 상황에서도** 인보이스 번호·날짜를 눈으로 바로 읽을 수 있습니다
- 오른쪽 끝 **NO. 03/20** 으로 라벨끼리 구분됩니다
- A4 세로 기준: 1열(QR 34mm, 5장/장) · 2열(QR 22mm, 16장/장)
- QR은 **벡터(SVG)** 라 축소 인쇄해도 스캔이 잘 됩니다
- 노란 하이라이트가 배경색이라 인쇄 옵션에서 "배경 그래픽"이 꺼져 있으면
  색만 빠질 수 있으나, **QR·글자·괘선은 그대로 인쇄**되어 식별에 지장 없습니다
